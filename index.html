Claw Key Manager

Gestão inteligente de API keys com health check automático, fallback e SQLite como fonte de verdade.

O problema
Keys expiram no meio da conversa

API keys atingem quota, expiram ou param de responder sem aviso. Trocar manualmente é irritante e quebra o fluxo.

A solução
Fallback automático

Monitoro todas as suas keys, detecto quando uma morre e ativo a próxima — sem você precisar fazer nada.

Funcionalidades

💚
Health Check

Testa todas as keys em intervalos de 1/2/5/10 min. Você sempre sabe qual está viva.

🔄
Fallback Automático

Se a key ativa morre, ativa a próxima viva automaticamente. Cooldown de 5 min anti-spam.

🗄️
SQLite

Banco local leve, sem dependências externas. Todas as keys e status em um só lugar.

🌐
Interface Web

Painel moderno com status em tempo real. Teste, ative, importe e exporte keys com um clique.

Stack

Flask SQLite Python 3 Docker Swarm Traefik
